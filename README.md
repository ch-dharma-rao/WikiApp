# WikiApp



Wikipedia App <br>

Features :
* Shows Random Articles 
* Shows Feartured Images
* Shows Categories
* Offline View

Tools, Architecture and Libraries : 
* Android Studio 
* Kotlin
* Retrofit
* Room/SQLite
* Glide
* MediaWiki Api
* MVVM Architecture
<br>
